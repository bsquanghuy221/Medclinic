Kết nối Supabase để dữ liệu tự động đồng bộ
trên mọi máy tính, điện thoại — miễn phí hoàn toàn

☁️ Supabase miễn phí — 500MB lưu trữ, không giới hạn request. Dễ cài hơn Firebase, không cần thẻ tín dụng.
1

Tạo tài khoản Supabase

Vào supabase.com → bấm Start your project → đăng nhập bằng GitHub hoặc email → bấm New project → đặt tên (VD: phongkham) → chọn region Southeast Asia (Singapore) → đặt mật khẩu → Create new project

2

Tạo bảng dữ liệu

Trong project → menu trái → SQL Editor → bấm New query → dán đoạn SQL bên dưới → bấm Run (F5)

create table patients (id text primary key, payload jsonb not null default '{}'::jsonb); create table visits (id text primary key, payload jsonb not null default '{}'::jsonb); alter table patients enable row level security; alter table visits enable row level security; create policy "public" on patients for all using (true) with check (true); create policy "public" on visits for all using (true) with check (true);
3

Lấy URL và API Key

Menu trái → Project Settings (biểu tượng bánh răng ⚙) → API → copy Project URL và anon / public key

4

Dán vào ô bên dưới và kết nối
